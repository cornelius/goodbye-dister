Dister, the SUSE Studio mascot, has served you well. SUSE Studio did 3.7 million builds for more than 800 thousand users. Now it's time to say goodbye. Dister is retiring. SUSE Studio has been shut down.

We have powerful alternatives for you:

Kiwi has been the core of SUSE Studio from the beginning. It's SUSE's command line image building tool. There you have full control. Use it to build your virtual machine and cloud images. It also does containers and installation images.

If you prefer a service to build your images, the Open Build Service is there for you. You get the full power of Kiwi plus automatic rebuilds on change of dependencies. This is your solution for continuous integration and delivery.

Going public cloud? SUSE's images are already there. Choose a SLES image for certifications and support. Or go with openSUSE for community and development.

If you moved to containers, Dister recommends to use the official openSUSE container images. They are a wonderful base for your container workloads.

Thanks for your support for all the years. We'll see you on the other side.
